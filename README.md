# Customized Momentum (nomadcoders challenge)

## Description

노마드 코더 챌린지
'바닐라JS로 크롬앱 만들기' / Momentum 클론 코딩

현재 시간, Todo list, 로그인, 현재 지역의 날씨 구현 


## About Project

### clock.js
- 실시간 시계 표시

### background.js
- 배경 이미지 랜덤 출력

### greeting.js
- 로컬 스토리지를 사용한 로그인

### quotes.js
- 랜덤 명언 출력

### todo.js
- 로컬 스토리지를 사용한 Todo list
- 등록, 삭제

### weather.js
- 날씨와 위치, 온도 (geolocation)
